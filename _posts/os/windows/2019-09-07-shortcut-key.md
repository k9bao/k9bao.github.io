# windows快捷键或命令

## 快捷键

- win + ←/→/↑/↓ 调整窗体大小，甚至隐藏。有时候找不到窗口了可以通过这个方法找出来
- win+Tab,  任务视图快捷键，显示当前运行的任务程序；
- win+Ctrl+D，  新建桌面窗口，可以重新再进行多个程序操作；
- win+tab 界面切换出多桌面模式,在这里可以增加删除桌面
- win+Ctrl+←/→，切换桌面窗口；

- ALT+F4 关闭当前窗口(光标必须在此窗口上)

## 常用命令

- start 打开新的窗口，如果跟.则打开当前目录资源管理器
- taskmgr.exe 任务管理器
