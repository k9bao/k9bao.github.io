# ipynb

## 简介

.ipynb文件即为Jupyter Notebook，是一个交互式笔记本，支持运行 40 多种编程语言。

　　Jupyter Notebook 的本质是一个 Web 应用程序，便于创建和共享文学化程序文档，支持实时代码，数学方程，可视化和 markdown。它的主要用途是：数据清理和转换，数值模拟，统计建模，机器学习等，Jupyter Notebook 与 IPython终端 共享同一个内核。
