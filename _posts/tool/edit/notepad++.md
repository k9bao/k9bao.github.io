# notepad++

## plugins

### NPP_HexEdit

[notepad](https://github.com/chcg/NPP_HexEdit/releases)
注意区分64位还是32位

解压缩后把HexEditor.dll文件复制到安装目录（如C:\Program Files\Notepad++\plugins），退出NotePad++重新打开即可！需要以十六进制显示时，点击菜单的：【插件】－【Hex- Editor】－【View in Hex】即可。
